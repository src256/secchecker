# Secchecker

Check passwords and other sensitive information.

## Installation


```ruby
gem 'secchecker'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install secchecker

## Usage

    $ gsecchecker ~/myrepository
